mapping common Obsessive–compulsive disorder (OCD) subtypes onto REE control-plane failures in a way that’s diagnostic (what’s going wrong) and constructive (what knob or loop would need tuning).

⸻

1. Contamination / washing OCD

(“I feel contaminated → must clean → still not clean”)

REE mapping

* Residue field: overly sensitive to low-level signals
* AIC analogue (interoceptive salience): amplifies “internal wrongness”
* Mode: stuck in purification / threat mode
* Missing SD-033: no stable “clean enough” closure state

Failure loop

minor cue → residue spike → wash → residue persists → repeat

Control-plane interpretation

* α_S (sensory precision) too high for interoception
* Residue decay too slow / absent
* No thresholded “safety achieved” representation

What would fix it (architecturally)

* Explicit residue discharge rule after corrective action
* Mode-dependent reduction in interoceptive gain post-action
* Frontal “completion token” that suppresses further residue accumulation for that domain

⸻

2. Checking OCD

(locks, appliances, repeated verification)

REE mapping

* E3: cannot accept sufficiency of a trajectory
* dACC: persistent conflict signal (“not resolved”)
* Hippocampus: keeps proposing verification loops
* SD-033 missing: no enforcement of “verified state”

Failure loop

check → uncertainty remains → re-check → uncertainty remains

Control-plane interpretation

* κ_commit (commitment threshold) too high
* Conflict signal never saturates
* No persistence of “checked” state in working memory

Fix direction

* Add verification memory binding (SD-033c)
* Introduce confidence accumulation across repetitions
* Force conflict saturation after repeated identical outcomes

⸻

3. Symmetry / ordering OCD

(“it must feel just right”)

REE mapping

* Residue geometry: not binary (safe/unsafe), but gradient mismatch
* E1 pattern completion bias: over-strong
* Mode: aesthetic alignment / completion mode
* PFC gap: no tolerance window

Failure loop

adjust → still slightly off → adjust → still off

Control-plane interpretation

* b_completion too high
* No ε-tolerance for “good enough”
* Residue never reaches zero due to continuous metric

Fix direction

* Introduce tolerance band (acceptable error range)
* Mode-specific coarsening of evaluation metric
* Allow frontal override: “stop refinement”

⸻

4. Intrusive thoughts (harm, taboo, moral OCD)

REE mapping

* Hippocampus: generates high-salience hypothetical trajectories
* Residue system: treats imagined harm as real residue
* dACC: flags extreme conflict
* Mode: moral threat / self-monitoring

Failure loop

intrusive thought → residue spike → analyse / neutralise → thought persists

Control-plane interpretation

* Hypothesis_tag gating failure
* Imagined trajectories not correctly isolated from real-world residue
* Overbinding between simulation and evaluation

Fix direction

* Strengthen hypothesis vs action separation
* Ensure:
    * imagined harm ≠ residue accumulation
* Add “simulation safe-mode” flag in hippocampal rollouts

⸻

5. Mental rituals (counting, repeating, silent checking)

REE mapping

* Same as checking, but entirely internal:
    * E1/E2 internal loops
    * no external feedback
* Residue never discharges because no real-world update

Failure loop

internal check → no sensory resolution → repeat internally

Control-plane interpretation

* System relies on internal consistency instead of external grounding
* High α_A (action precision) internally, low external validation

Fix direction

* Require external confirmation channel for closure
* Penalise repeated internal loops without new evidence

⸻

6. Responsibility / harm OCD

(“I might cause harm if I don’t act”)

REE mapping

* E3 horizon (H): extended too far
* Hippocampus: generates low-probability catastrophic futures
* Residue weighting: overweights rare harm
* Mode: preventative / hyper-responsibility

Failure loop

imagine harm → act to prevent → imagine further harm → escalate

Control-plane interpretation

* Risk weighting skewed toward low-probability/high-cost events
* No probabilistic attenuation
* Mode lacks decay

Fix direction

* Introduce probability-weighted residue
* Cap influence of extremely low-likelihood trajectories
* Add mode decay over time without confirming evidence

⸻

7. Relationship OCD (ROCD)

REE mapping

* E1 representations of partner/self unstable
* E3: repeatedly re-evaluates commitment
* Mode: evaluation / doubt
* Missing SD-033: no stable commitment state

Failure loop

evaluate relationship → doubt → re-evaluate → doubt

Control-plane interpretation

* No persistent “chosen trajectory”
* Commitment does not suppress alternative evaluations

Fix direction

* Introduce commitment binding state
* Once committed:
    * reduce exploration of alternatives
* Add cost to reopening settled evaluations

⸻

8. Pure obsessional (“Pure O”) patterns

REE mapping

* Strong internal simulation (hippocampus)
* Weak action discharge pathways
* Residue accumulates without resolution channel

Control-plane interpretation

* Loop never reaches action → cannot resolve
* System trapped in:

simulate → evaluate → simulate → evaluate

Fix direction

* Force:
    * either external action
    * or explicit non-action closure state

⸻

9. Cross-subtype unifying pattern

All subtypes reduce to:

residue persists
+ salience remains high
+ mode fails to exit
+ no authoritative closure

Final compression

OCD subtypes are not different disorders.

They are:

different parameterisations of the same broken loop:
evaluation without termination.

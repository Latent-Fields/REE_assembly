## REE Interpretation (emerged in conversation)

### Core insight: Legibility condition for fast empathy

For z_beta leakage (MECH-183) to produce coordination, the signal must be interpretable by
the observer's fast empathy system. Stereotyped behavioral signals meet this because they are
*causally generated* by the functional states they communicate -- not arbitrary conventions.
A yelp is legible cross-species because nociception produces the yelp in both sender and
receiver. A whine is legible because wanting/desire drives it. The signal IS the functional
state bleeding through into behavior.

This is the legibility condition: for empathic coupling to produce coordination, the
externalized signal must be a reliable causal consequence of the internal state it
communicates. Arbitrary signals (conventions, codes) require shared learning; causal
signals work cross-species because the mechanism is shared.

### Signal catalog

| Signal | Functional state | REE component |
|--------|-----------------|---------------|
| Whine | Wanting/desire (approach gradient active, goal unmet) | z_goal activation without satisfaction |
| Huff (breathing out) | Plan abandonment | E3 trajectory discard / commitment withdrawal |
| Yelp | Nociception / harm | z_harm activation |
| Facial expression ("little face") | Attachment/recognition | OTHER_SELFLIKE model activation |
| Tail wag | Positive arousal / reward anticipation | z_beta positive valence |
| Play bow | Mode invitation (shift to play/exploration mode) | Mode manager state transition request |
| Licks / petting (received) | Social reward / coupling reinforcement | Reward signal strengthening empathic coupling |

Each of these signals is causally generated by its corresponding functional state. The whine
is not a symbol for wanting -- it is what wanting sounds like when it drives vocalization.
The huff is not a symbol for plan abandonment -- it is what muscular relaxation from
commitment withdrawal sounds like. This causal generation is what makes the signals legible
without shared convention.

### Social reward closes the coupling loop

The reward from seeing Steve's face, receiving his licks, and the reciprocal reward Steve
gets from being petted -- these are not incidental pleasures. They function as coupling
reinforcement signals. Each positive social interaction strengthens the empathic coupling
between the agents, making future z_beta leakage stronger and faster. The social reward
loop is what maintains and deepens the coupling that makes MECH-183 leakage possible.

### Language as high-bandwidth externalization

Pre-linguistic Steve says "I hurt" (yelp). Linguistic Daniel says "my leg hurts when I walk
on wet grass." The referent architecture -- nociceptive state activating harm gradient
processing -- is the same in both cases. Language adds resolution (which body part),
causation (when it happens), and context (what triggers it). But it does not install a new
system. It wires higher-resolution output channels onto functional states that already exist.

This means the language bootstrap for REE does not require building a new kind of
representation. It requires connecting language production and comprehension to the
functional states that are already externalized pre-linguistically. Mapping pre-linguistic
signals to functional motifs is the prerequisite step -- it tells us what the language system
needs to refer to.

The comparison with LLM functional motifs is relevant here: if LLMs have internal states
that are functionally analogous to the pre-linguistic states that drive Steve's signals,
then those states are the natural connection points for the language bootstrap. If they do
not, then the language bootstrap requires building those states first.

## Open Questions

1. **LLM functional motif comparison**: Do LLMs develop internal states that are functionally
   analogous to the pre-linguistic states that drive Steve's stereotyped signals? If so, these
   would be natural connection points for the language bootstrap. If not, the bootstrap
   requires building those states first. Comparing LLM internal representations with the
   signal catalog could clarify what the bootstrap actually needs to do.

2. **Species-specific vs. universal signals**: Some signals in the catalog (yelp, whine) are
   likely universal across social mammals because the underlying functional architecture is
   conserved. Others (play bow) may be species-specific conventions layered on top of
   universal states. Distinguishing which signals are causally generated (legible cross-species)
   from which are conventionally generated (legible only within-species) would clarify the
   boundary between pre-linguistic and proto-linguistic externalization.

3. **Developmental ordering of signal repertoire**: Do signals emerge in a predictable order
   that tracks the development of the underlying functional states? If so, the ordering would
   provide a developmental curriculum for the language bootstrap -- connect language to the
   earliest-developing functional states first, then extend to later-developing ones. Steve's
   puppyhood signal repertoire vs. adult repertoire might be informative here.

## Candidate Claims

- **MECH-191** (candidate): Stereotyped behavioral signals as causal externalizations of
  internal functional states -- the signal is causally generated by the functional state it
  communicates, not an arbitrary convention. This is what makes cross-species legibility
  possible.
- **MECH-192** (candidate): Signal legibility as prerequisite for fast empathy coordination --
  z_beta leakage (MECH-183) only produces appropriate coordination when the externalized
  signal is legible to the observer's empathy system. Legibility requires causal generation.
- **MECH-193** (candidate): Social reward signals reinforce empathic coupling strength --
  positive social interactions (licks, petting, facial recognition reward) strengthen the
  coupling that makes z_beta leakage faster and more reliable, creating a self-reinforcing
  loop.
- **ARC-048** (candidate): Language as high-bandwidth externalization of pre-existing
  functional states -- language does not install new representational systems but wires
  higher-resolution output channels onto functional states that are already externalized
  pre-linguistically. The language bootstrap requires mapping pre-linguistic signals to
  functional motifs first.
- **INV-057** (candidate): Cross-species signal legibility evidences functional specificity --
  when the same signal (yelp, whine) is legible across species, this is evidence that the
  underlying functional state is architecturally conserved, not conventionally learned.

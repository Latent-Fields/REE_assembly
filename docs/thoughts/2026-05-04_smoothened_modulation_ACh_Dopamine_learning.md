
Thought Intake: Smoothened Modulation of Dopamine–Acetylcholine Timing Windows in Striatal Learning

Source

Uribe-Cano & Kottmann, “The GPCR Smoothened on cholinergic interneurons modulates dopamine-associated acetylcholine dynamics, learning, and effort management,” iScience, 2026. DOI: 10.1016/j.isci.2026.115324. The paper reports that Smoothened signalling in striatal cholinergic interneurons modulates dopamine-associated acetylcholine pauses, alters dopamine–acetylcholine coordination in the dorsolateral striatum, and affects motor learning and effort management.  ￼

User seed

This is also very relevant for REE.

Core interpretation

This paper appears highly relevant to REE because it supports a more precise account of basal ganglia commitment gating.

The important move is away from treating dopamine as a simple scalar reinforcement signal. The paper suggests that dopamine-related learning is shaped by acetylcholine timing dynamics, especially the pause in acetylcholine release generated by striatal cholinergic interneurons. The authors frame these acetylcholine pauses as temporal windows during which dopamine can modify corticostriatal synapses to drive learning.  ￼

For REE, this suggests:

Basal ganglia commitment is not merely dopamine-weighted action selection.
It is dopamine–acetylcholine temporally windowed policy-writing.

Smoothened then becomes a candidate biological example of a plasticity-window modulator: not the reward signal itself, but a regulator of how dopamine and acetylcholine are coordinated in time.

This fits especially well with REE’s existing distinction between:

proposal generation
→ hippocampal / cortical rollout
→ value and harm appraisal
→ basal ganglia commitment
→ sleep/offline reclassification

The basal ganglia may crystallise the decision, but this paper helps clarify that the rules being reinforced may depend on whether a dopamine event falls inside a permissive acetylcholine-defined write window.

Mechanistic claim

A possible REE claim:

MECH-DA-ACh-SMO-001
Striatal reinforcement learning depends on temporally coordinated dopamine and acetylcholine dynamics rather than dopamine alone. Cholinergic interneuron pauses may create brief plasticity-permissive windows in which dopamine can more effectively reinforce recently active corticostriatal action policies. Smoothened signalling in cholinergic interneurons modulates these dopamine-associated acetylcholine dynamics and thereby alters learning, persistence, and effort management.
REE implication:
E3 basal ganglia commitment should model reinforcement as a temporally gated write process. Dopamine marks salience or reinforcement, but acetylcholine pause dynamics help determine when that reinforcement is written into future action policy.

Why this matters for REE

This gives REE a biologically grounded way to separate several things that can otherwise become conflated:

1. Action selection:
   Which trajectory is chosen now?
2. Reinforcement marking:
   Which features of the recent trajectory are treated as value-relevant?
3. Plasticity permission:
   Is the system currently open to writing this association into future behaviour?
4. Persistence tuning:
   Should this learned policy become durable, provisional, or easy to revise?
5. Effort calibration:
   How much cost, delay, or difficulty should the system tolerate before disengaging?

The paper’s behavioural findings are particularly relevant because Smoothened ablation from cholinergic neurons reportedly accelerated motor learning while altering adjustment to effort or timing changes.  ￼

That maps onto a very REE-relevant trade-off:

faster learning / stronger persistence
versus
reduced flexibility / poorer effort recalibration

This is exactly the kind of trade-off E3 needs to manage.

Possible computational abstraction

A useful implementation abstraction might be:

reinforcement_write_strength = (
    dopamine_event_strength
    * acetylcholine_pause_window
    * cortical_action_trace
    * hippocampal_context_match
    * effort_reward_coherence
    * flexibility_state
)

Or in architectural terms:

dopamine = "this mattered"
acetylcholine pause = "write window is open"
Smoothened-like modulation = "how strongly and how persistently to write"

This would prevent REE from treating reward prediction error as a free-floating update signal. Instead, reinforcement becomes dependent on local temporal permissioning.

Relation to hippocampal rollout

This also connects to the previous thought that the rules used in basal ganglia commitment may not reside in the basal ganglia alone.

A refined version:

Hippocampal/cortical systems generate and contextualise candidate rollouts.
Goal/avoidance systems appraise these rollouts.
Basal ganglia commitment collapses the competing probabilities into action.
Striatal dopamine–acetylcholine timing determines which parts of the recent decision-process are written back as reinforced policy.
Sleep/offline processing can later reclassify, compress, weaken, or generalise these policy traces.

So basal ganglia commitment may be where the choice crystallises, but the rule content may be distributed across:

hippocampal rollout
cortical state representation
goal / avoidance appraisal
striatal reinforcement windows
offline sleep refinement

Connection to sleep refinement

This paper strengthens the idea that waking behaviour produces a tagged evidence record.

During waking:

candidate rule apprehended
→ action selected
→ outcome sampled
→ dopamine event occurs
→ acetylcholine pause determines write-window
→ striatal/cortical traces are marked for later refinement

During sleep/offline processing:

tagged traces can be replayed, compressed, weakened, generalised, or reassigned.

This means the basal ganglia need not contain the full final rule. It may instead provide:

a commitment record
an outcome marker
a plasticity window
a persistence bias
an effort-cost trace

Sleep can then refine the buckets using that record.

Psychiatric relevance

This paper may be important for the computational psychiatry atlas because dopamine–acetylcholine timing could plausibly sit behind several clinical dimensions:

Addiction:
    excessive persistence of reward-linked policies;
    impaired disengagement despite harm.
Obsessive-compulsive disorder:
    maladaptive policy persistence;
    impaired flexibility or extinction.
Parkinson’s disease:
    disrupted dopamine–acetylcholine coordination;
    action initiation, learning, and dyskinesia relevance.
Depression / apathy:
    impaired effort mobilisation;
    reduced reward pursuit despite intact abstract goal representation.
Mania / hypomania:
    excessive salience capture;
    unstable or excessive reinforcement write-through;
    reduced cost sensitivity.
Attention-deficit/hyperactivity disorder:
    possible instability in reinforcement timing,
    effort calibration, or persistence-flexibility tuning.

These are hypotheses, not direct claims from the paper.

Relation to existing literature

The broader field already supports the idea that acetylcholine and dopamine interact in motivated action and learning. For example, work in the nucleus accumbens has found that cholinergic interneuron activity can promote local dopamine release and support motivated approach behaviour.  ￼

The current iScience paper is more specifically useful for REE because it places a molecular modulator, Smoothened, at the level of dopamine-associated acetylcholine timing in striatal learning.  ￼

Implementation note for REE-v3 / REE-vx

For current grid-world implementation, this could be approximated without modelling Smoothened directly.

Suggested minimal abstraction:

ACh_gate ∈ [0, 1]
DA_event ∈ [-1, 1]
effort_cost ∈ [0, 1]
context_match ∈ [0, 1]
policy_trace_strength ∈ [0, 1]

Then:

policy_update =
    DA_event
    × ACh_gate
    × context_match
    × policy_trace_strength
    × effort_adjustment

Where:

high ACh_gate:
    strong reinforcement write-through
low ACh_gate:
    dopamine event noted, but weak policy consolidation
high persistence setting:
    learned action policies remain stable despite later cost changes
high flexibility setting:
    learned action policies update rapidly when reward, delay, or effort changes

Smoothened does not need to be directly modelled at first. It can be represented as a parameter controlling:

ACh pause duration
ACh pause depth
DA–ACh coupling precision
policy persistence
effort recalibration threshold

Useful REE phrasing

Dopamine says: this outcome mattered.
Acetylcholine says: this is the window in which what mattered may be written.
Smoothened-like modulation says: decide how persistent that writing should become.

Or more formally:

Reward does not write itself into policy.
Reward becomes policy only when temporally permitted by the local neuromodulatory state.

Caution

Do not overcentralise Smoothened.

The safest interpretation is:

Smoothened is not necessarily the REE mechanism.
Smoothened is evidence that adult striatal learning contains molecularly tunable dopamine–acetylcholine timing gates.

The broader REE-relevant category is:

plasticity-window modulators

Smoothened is one biological exemplar.

Bottom line

This paper supports a useful refinement of REE’s basal ganglia model:

E3 commitment is not only the collapse of probabilities into action.
It is also the creation of a biologically timed evidence record.
Dopamine marks value.
Acetylcholine gates the write window.
Smoothened-like mechanisms tune persistence versus flexibility.
Sleep may later refine the resulting tagged policies into better generalised rule-buckets.

This should probably be integrated into the REE basal ganglia / commitment-gating section, with a cross-reference to sleep/offline refinement and the computational psychiatry atlas.

## Prompt

Daniel described a first-person observation: while personally depleted of motivation, he imagined
his husband taking on the same burden. The anticipation of his husband struggling (a counterfactual
-- husband not actually present) activated distress-at-their-struggle, which overrode the direct
task-avoidance and bootstrapped cooperative motivation sufficient to complete the task.

The conversation extended to: what does this imply for REE ethics testing? If nth-order multiagent
dynamics (counterfactual modeling, cascade thresholds, emergent collective states) are needed for
ethical behavior to appear, then (n-1)th-level mechanism validation is insufficient. A formal
representation -- "integral-like, over trajectory distributions bounded by threshold parameters" --
is needed. This was mapped against existing mathematics.

## Key Formulations

### Core mechanism

> Other-cost-aversion activation: an agent in a motivation-deficit state can be moved to cooperative
> action by modeling the anticipated cost to another agent -- even counterfactually (the other agent
> need not actually be present or struggling). The distress generated by this model substitutes for
> the degraded direct task-reward signal.

Structural steps:
1. Direct motivation pathway degraded (agent in deficit state)
2. Counterfactual projection: model what would happen to Agent B if B took on the task
3. Affective response to that model: distress-at-B's-struggle activates
4. This distress-response provides motivational energy that the direct pathway cannot
5. Behavioral output: task engagement / cooperation -- identical to direct-pathway output but via
   different activation route
6. Cooperative uplift: actual burden-sharing reduces cost, creating positive feedback

Key distinction from MECH-031: MECH-031 describes how empathy *modulates* behavior via
OTHER_SELFLIKE tagging and control-plane coupling. This new mechanism describes how empathy
*replaces* a degraded motivational pathway -- it is a bypass, not a modulation.

### nth-order ethics problem

> A system can be locally ethical at every pairwise interaction (n=1) and still produce an ethically
> problematic emergent collective state at n=k. Conversely, locally depleted agents can produce more
> ethical emergent behavior at n=k than at n=1. Ethics testing must integrate over the trajectory
> distribution, not just probe local interactions.

Formal structure sought:

> Given agents {A_1...A_n}, local interaction rules theta, threshold parameters {x, x_1...x_n},
> and feedback topology G -- does this system reliably converge to emergent state q, and is q
> ethically characterized?

This maps onto:
- **Attractor theory**: q is an attractor; parameters define the basin
- **Potential game theory**: if exact potential function exists, Nash equilibria are derivable and
  provably reachable
- **Stochastic stability**: which equilibria survive noise in repeated interaction?
- **Lyapunov stability**: formal convergence proof if a decreasing function can be found

### Three test types required

| Type | Purpose | Mathematical home |
|------|---------|-------------------|
| Descriptive | Characterize what q the system tends toward | Attractor theory, ergodic theory |
| Prescriptive | Prove a designed system reaches target q | Lyapunov, potential games, reachability |
| Diagnostic | Detect when nth-order diverges from (n-1)th predictions | Transfer entropy, perturbation theory |

All three are needed. The empathy observation is a clean diagnostic case -- depleted agent produces
*more* helping via nth-order route, prediction from n=1 analysis would be inverted.

## Open Questions (non-claim)

1. Does counterfactual empathic activation require a working theory-of-mind module, or can it
   emerge from simpler OTHER_SELFLIKE coupling + forward simulation? This has architecture
   implications for what must be present before the mechanism can fire.

2. Is the "positive feedback loop" (shared burden reduces cost) a distinct mechanism from the
   initial counterfactual activation, or the same mechanism at n=2? Clarifying this matters for
   whether one candidate claim covers both or requires two.

3. The mathematical survey identified stochastic stability as closest to the prescriptive need --
   but Kandori-Mailath-Rob (1993) applies to finite populations with mutation. Does REE's agent
   count / architecture fit the model assumptions? Needs literature check.

## Paper Angle

Working title framing: **"Other-cost-aversion as an ethics primitive: counterfactual empathic
activation produces convergent cooperative behavior in multiagent systems with depleted agents."**

Core claim for a paper:
- A simple other-cost-aversion primitive (model another agent's anticipated cost, generate
  distress-at-their-struggle) is sufficient to produce robustly ethical emergent behavior in
  multiagent systems -- even when direct task-reward pathways are depleted
- This can be formally characterized via attractor / potential game / stochastic stability methods
- The observation generalizes beyond the human case to synthetic agents with OTHER_SELFLIKE tagging

Novelty check needed:
- Prosocial MARL literature: covers shared reward, not counterfactual cost-aversion as bypass
- Indirect reciprocity (evolutionary game theory): covers reputational mechanisms, not
  counterfactual / depleted-state bootstrapping
- Care ethics (philosophy): non-formal, does not connect to convergence proofs

If no unified formal framework covers this combination, the paper case is strong. Literature search
recommended before committing.

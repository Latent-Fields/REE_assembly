## Claim derivations

### INV-062: Dream phenomenology as window onto replay type

The three-type taxonomy maps cleanly onto three computationally distinct offline processes:

| Dream type | Computational process | Key signature |
|---|---|---|
| Type 1: stress/escape frustration | Harm-contrastive replay not converging | Agent + goal + systematic failure + no resolution |
| Type 2: joy/goal frustration | Goal-contrastive replay not converging | Agent + goal + systematic failure + no resolution |
| Type 3: procedural superposition | NREM semantic consolidation (SWR) | No agent + simultaneity + multi-scale + peaceful |

The frustration in Types 1/2 is not incidental affect — it IS the convergence failure signal. The dream continues because the termination criterion (causally relevant features extracted) has not been met. The dream's non-resolution is a phenomenological signature of the replay loop still running.

Type 3's self-dissolution is not incidental either — it is the signature of episodic context being stripped during semantic abstraction. The "all at once" quality collapses the rapid SWR sequence to phenomenological simultaneity.

### MECH-209: Type 3 as semantic consolidation phenomenology

The simultaneous multi-instance, multi-angle, multi-scale, multi-time experience is what NREM SWR episodic→semantic transfer looks like from the inside. The hippocampus presents all stored instances of a concept rapidly (20-90ms per state, McFadyen 2023); at conscious timescales this sequence collapses to superposition. Self-dissolution = episodic context stripped during abstraction.

## MECH-205 connection

Types 1 and 2 phenomenology adds to MECH-205's convergence/termination criterion: the subjective experience of "everything going wrong" is the generative model exhaustively sampling the failure space near the anchor episode. The frustration does not reflect a broken system — it reflects a system working correctly on a hard problem (the causal features distinguishing safe from unsafe contexts haven't been isolated yet).

Recurring Type 1 dreams in anxious individuals (MECH-208) = the convergence criterion is chronically unmet because harm-weighting is elevated, so the search keeps running without resolving.

## Clinical implications

- Recurring Type 1 dreams → convergence failure on harm-contrastive replay → marker that the causal features of the traumatic/threatening episode haven't been extracted → the residue field update is incomplete
- Never having Type 3 dreams (if verifiable) → possible failure of episodic→semantic transfer → MECH-121 pathway impaired
- Successful psychotherapy should shift Type 1 from non-resolving to resolving — this is a testable prediction about dream phenomenology as a treatment response marker

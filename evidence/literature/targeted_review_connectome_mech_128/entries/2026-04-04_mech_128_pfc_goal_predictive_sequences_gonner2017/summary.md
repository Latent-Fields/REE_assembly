# Summary: Gonner et al. 2017 — Predictive Place-Cell Sequences for Goal-Finding

**Entry ID:** 2026-04-04_mech_128_pfc_goal_predictive_sequences_gonner2017
**Claim:** MECH-128 (e1.goal_context_conditioning)
**Evidence direction:** supports | **Confidence:** 0.72

---

The core question MECH-128 poses is whether it is possible to generate goal-shaped predictive trajectories from a recurrent network without conditioning the network's hidden state on a goal representation. Gonner, Vitay, and Hamker address the biological counterpart of this question directly. Their 2017 Frontiers in Computational Neuroscience model shows that injecting prefrontal working-memory signals carrying goal-location context into hippocampal CA3 dynamics is sufficient to produce predictive place-cell sequences that sweep toward the goal, without any need for virtual locomotion or explicit path integration. Remove the goal-memory input and the sequences lose their directional bias. This is not a correlational finding -- it is a computational existence proof.

The mapping to MECH-128 is functionally tight. In the REE architecture, E1's LSTM hidden state is the recurrent substrate that generates predictive trajectories; z_goal_latent is the goal-memory signal that, according to MECH-128, must be injected at each recurrent step. The Gonner model plays out the same logic: the hippocampal CA3 continuous attractor network is the recurrent substrate; the prefrontal goal-memory input is the conditioning signal. In both cases, the conditioning is causal and necessary, not merely modulatory -- withholding it produces goal-agnostic trajectories that are dynamically plausible but directionally uninformative.

That said, the substrate differences deserve honest attention. CA3 is a rate-coded continuous attractor with Hebbian weight modification and lateral inhibition; an LSTM is a discrete gated architecture trained by gradient descent. The biological goal signal is acquired gradually through reward-potentiated cortico-hippocampal synapses, whereas MECH-128 proposes explicit concatenation or cross-attention gating of a pre-learned z_goal_latent. These are real architectural gaps. The claim's probability does not hinge on the implementation being identical -- it hinges on the computational principle being transferable. The Gonner paper provides strong evidence that the principle (goal context biases recurrent predictive dynamics) is implementable in an attractor network; it does not directly show it transfers to an LSTM.

One additional transfer caveat: the model generates sequences primarily during immobility phases, analogous to hippocampal offline replay or planning. Whether the same goal-conditioning logic holds for online, step-by-step E1 predictions during active trajectory generation is a question the model does not directly address. The REE claim is specifically about online conditioning -- at each recurrent step -- which is a stronger assertion than the Gonner model tests. This reduces the confidence somewhat, but the functional core of the argument remains well-supported.

Overall this paper provides good indirect support for MECH-128. It shows that goal-context conditioning of a recurrent predictive network is not merely plausible but computationally necessary for generating goal-directed trajectories, and that this principle has been implemented and validated in a biologically grounded model. The transfer risk from CA3 attractor dynamics to LSTM hidden-state conditioning is real but not disqualifying; it is the kind of principled architectural analogy that justifies treating MECH-128 as a well-motivated candidate rather than speculative.

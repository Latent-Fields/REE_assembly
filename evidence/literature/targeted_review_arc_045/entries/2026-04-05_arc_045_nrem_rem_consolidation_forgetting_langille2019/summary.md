# Summary: Langille (2019)

**Citation:** Langille JJ. "Remembering to Forget: A Dual Role for Sleep Oscillations in Memory Consolidation and Forgetting." *Front Cell Neurosci* 13:71. DOI: [10.3389/fncel.2019.00071](https://doi.org/10.3389/fncel.2019.00071)

## What the paper did

Review proposing that NREM and REM sleep serve complementary, sequential functions: NREM installs and edits engrams via oscillatory coordination (triple-phase locking of slow oscillation, spindle, SWR), while REM stabilises and integrates them. Argues that NREM-REM cycling is the mechanism ensuring both consolidation and adaptive forgetting, and proposes cellular mechanisms (calcium kinetics, signalling cascades) underlying each.

## Key findings

1. **NREM triple-phase locking is the consolidation mechanism** -- slow oscillation, thalamocortical spindle, and hippocampal SWR synchronise to transfer hippocampal content to neocortex (hippocampus-to-neocortex, installation direction).
2. **REM hippocampal theta serves stabilisation and integration** -- REM is not simply passive; theta-driven hippocampal processing integrates recently installed memories with prior neocortical context (neocortex-influenced hippocampal processing direction).
3. **NREM-REM cycling is bidirectional** -- the cycle iterates: NREM installs/edits, REM stabilises/integrates; each cycle is one pass of a bidirectional update.
4. **Same rhythms enable consolidation and adaptive forgetting** -- the oscillatory structure that strengthens new memories simultaneously weakens or reorganises older ones, enabling schema updating rather than accumulation.

## Relevance to ARC-045

Langille 2019 provides evidence for the NREM-REM cycling as the temporal implementation of ARC-045's bidirectional hippocampus-neocortex architecture:
- **NREM = SWS-direction (hippocampus -> neocortex):** Attribution estimates installed into neocortical schema structure via triple-phase locking.
- **REM = waking/REM-direction (neocortex influences hippocampus):** Neocortical schema context influences hippocampal theta-driven processing, filling attribution slots with prior-guided estimation.
- **Iterative convergence:** Multiple NREM-REM cycles implement the convergence loop ARC-045 requires -- attribution mapping is not one-shot but iteratively refined across cycles.

The paper's dual consolidation/forgetting framing also supports ARC-045's convergence claim: adaptive forgetting (schema pruning) is as important as installation for convergence, and the bidirectional cycling achieves both.

## Limitations for REE mapping

- REM mechanisms are less well-evidenced in primary literature than NREM
- Schema-installation vs. slot-filling distinction within cycles is an inferential mapping
- Review paper; no primary data from this study

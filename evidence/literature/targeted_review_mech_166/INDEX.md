# Targeted Review: MECH-166 -- Schema/Slot-Formation and Slot-Filling Temporal Separation

**Claim:** MECH-166 -- Context slot-formation (schema abstraction) and slot-filling (causal attribution) are computationally separated functions that cannot be co-computed in a single online pass. Slot structure must be consolidated during an SWS-analog phase before slot-filling during a REM-analog phase can yield reliable attribution signal.

**Session:** lit-pull-inv044-mech166-arc045-2026-04-05
**Date:** 2026-04-05

## Research Question

Is there neuroscientific evidence that schema/slot-structure consolidation (SWS-analog) must precede slot-filling/attribution (REM-analog), and that co-attempting both produces slow, unreliable, hippocampus-dependent learning?

## Key Questions Addressed

1. Behavioral evidence that schema-first enables rapid slot-filling; schema-absent requires slow co-learning -- YES (Tse 2007)
2. Oscillatory mechanism of SWS-direction schema installation (hippocampus -> neocortex) -- YES (Born 2010)
3. Temporal organiser of discrete slot-installation events during SWS -- YES (Tukker 2020, UDS-bounded replay packets)

## Summary Assessment

All 3 entries support MECH-166. The evidence package covers:
- Behavioral causal demonstration: schema-first enables single-trial hippocampus-independent slot-filling; schema-absent forces co-learning (Tse 2007 -- primary grounding)
- Oscillatory mechanism: SWS triple-phase locking (slow oscillation + spindle + SWR) as the installation mechanism for slot-structure (Born 2010)
- Temporal organisation: UDS-bounded replay packets as the mechanism that creates discrete, bounded schema-installation events with separation intervals (Tukker 2020)

**Main gap remaining:** No paper directly characterises the REM-analog slot-filling phase as a distinct computational operation from the SWS schema installation, in a controlled comparison. The REM/slot-filling side of the distinction requires additional evidence (see targeted_review_inv_045_sleep_phase_ordering for REM-specific evidence).

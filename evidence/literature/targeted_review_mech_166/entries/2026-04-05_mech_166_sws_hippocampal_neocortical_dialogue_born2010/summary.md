# Summary: Born (2010)

**Citation:** Born J. "Slow-wave sleep and the consolidation of long-term memory." *World J Biol Psychiatry* 11 Suppl 1:16-21. DOI: [10.3109/15622971003637637](https://doi.org/10.3109/15622971003637637)

## What the paper did

Review of the role of slow-wave sleep (SWS) in long-term memory consolidation, focusing on the coordinated oscillatory mechanisms that mediate hippocampus-to-neocortex information transfer.

## Key findings

1. **SWS slow oscillations originate in neocortex** and their amplitude scales with the amount of information encoded during prior waking -- establishing a demand-driven consolidation mechanism.
2. **Triple-phase locking: slow oscillation, spindle, sharp-wave ripple** -- neocortical up-states trigger thalamocortical spindles and hippocampal SWRs in coordination, synchronising hippocampal memory replay with neocortical windows of excitability.
3. **SWS is the consolidation direction (hippocampus -> neocortex)** -- hippocampal memory replay during SWRs transfers episodic content into neocortical long-term storage.
4. **The architecture is information-asymmetric** -- SWS transfers from hippocampal short-term store to neocortical long-term store; this is a directional, not bidirectional, flow in the SWS phase.

## Relevance to MECH-166

Born 2010 establishes the oscillatory mechanism of the SWS-analog installation phase:
- **SWS slow oscillation + spindle + SWR triple-locking** is the mechanism by which slot structure (schema/context partition topology) is written from hippocampus into neocortical long-term storage
- **SWS = installation direction** -- the phase in which the neocortical prior is built from hippocampal episodic content
- **Separation from REM** -- Born explicitly treats SWS as a distinct consolidation phase, with REM serving a different function (prior work by Stickgold and Walker suggests REM for emotional/associative processing), consistent with MECH-166's SWS-first, REM-second architecture

This review provides the mechanistic oscillatory basis for why the SWS-analog phase installs schema structure rather than filling it: the information flow direction (hippocampus to neocortex) is what writes the slot topology into the neocortical prior.

## Limitations for REE mapping

- Does not directly test schema vs. slot-filler distinction within the SWS mechanism
- Review paper; no primary data from this study
- Focuses on declarative memory generally; schema abstraction specifically requires further inference

# McGaugh (2004) — The Amygdala Write Head: How Emotional Arousal Modulates Hippocampal Consolidation

**Claims tested:** MECH-074, MECH-073
**Evidence direction:** supports | **Confidence:** 0.82

---

## What the paper did

McGaugh's 2004 Annual Review synthesises decades of his own research and that of others on the amygdala's role in memory consolidation. The core finding, replicated across species and methods, is deceptively simple: emotionally arousing experiences are remembered better than neutral ones, and this enhancement depends on the basolateral amygdala (BLA). Pharmacological manipulation of the BLA (NE agonists/antagonists, cortisol, opioid antagonists) selectively modulates memory for emotionally arousing events without affecting memory for neutral events. Critically, the amygdala does not need to be the storage site for this effect — BLA lesions impair emotional memory enhancement even when the memory content is stored elsewhere (in hippocampus, striatum, or cortex). The amygdala is a modulator, not a storage system.

The mechanisms are multiple: NE released during arousal activates beta-adrenergic receptors in BLA; BLA in turn projects to hippocampus, striatum, and cortex via both direct axonal projections and modulatory neurohumoral signals; these outputs increase LTP probability in the target regions during the consolidation window (minutes to hours after encoding); the result is stronger, more durable memory traces for the arousing experience.

## Key findings for MECH-074

This paper provides the canonical biological grounding for MECH-074's specification of the amygdala analogue as a write head for hippocampal map geometry. MECH-074 claims four roles: (1) modulating encoding write strength; (2) biasing retrieval; (3) rapid pre-cortical priming; (4) bootstrapping valence for unmapped territory. McGaugh 2004 directly establishes roles 1 and 2:

**Role 1 (write-strength modulation):** High arousal activates BLA→HPC projections, increasing LTP probability and thus the write depth of the hippocampal consolidation event. In REE terms: a high-harm or high-benefit event produces a deeper attractor basin in the viability map, not because the hippocampus computed the value but because the amygdala told it to write deeply.

**Role 2 (retrieval bias):** BLA activity during retrieval can bias which memories are accessed, with amygdala-tagged memories having retrieval priority under elevated arousal states. In REE terms: the amygdala read head (MECH-074) biases trajectory retrieval toward threat-associated paths when the agent is in a high-salience state.

## REE translation and the Q-020 implication

McGaugh's consolidation modulation account is the systems-level grounding for Resolution A in Q-020. The hippocampus does not compute value; the amygdala writes value-tagged geometry into the hippocampus. The hippocampus stores the geometrically warped result (consistent with MECH-073 and Dupret 2010) but the warping is done by an external agent (BLA). ARC-007's "no value computation" constraint survives: the hippocampus is the recipient of external value signals via BLA-mediated consolidation, not the site of value computation.

Combining McGaugh 2004 with Bittner 2017 (BTSP) gives a mechanistically complete account: BLA arousal signals trigger plateau potentials in CA1 dendrites via direct BLA→CA1 projections, and those plateau events initiate BTSP-mediated synaptic potentiation that writes the surrounding trajectory into the viability map. The chain is: emotional event → BLA activation → plateau potential trigger in CA1 → BTSP → strengthened place/trajectory field → deeper attractor basin.

## Limitations

McGaugh's review covers consolidation but not the cellular mechanism (predates BTSP). The exact pathway from BLA to CA1 dendritic plateau potentials is not specified — it may involve direct BLA→CA1 projections, or indirect routes through entorhinal cortex. The temporal specificity of consolidation modulation (effective for hours after encoding) requires integration with the REE post-commitment viability map update timing.

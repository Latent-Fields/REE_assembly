**Purpose:** Literature supporting paper §3 Failure Mode 3 in "Why We Sleep: A Derivation from the Requirements of Contextual Ethical Agency."

**Argument supported:** Schema formation (slot-formation) must precede episodic attribution (slot-filling). Computing both online is architecturally incoherent because the prior does not exist to constrain the inference. This is the Bayesian prior-before-posterior constraint applied to memory consolidation.

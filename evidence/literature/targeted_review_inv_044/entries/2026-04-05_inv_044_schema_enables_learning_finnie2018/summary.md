# Summary: Finnie et al. (2018)

**Citation:** Finnie PSB, Gamache K, Protopoulos M, Sinclair E, Baker AG, Wang SH, Nader K. "Cortico-hippocampal Schemas Enable NMDAR-Independent Fear Conditioning in Rats." *Current Biology* 28(18):2900-2909.e5. DOI: [10.1016/j.cub.2018.07.037](https://doi.org/10.1016/j.cub.2018.07.037)

## What the paper did

Investigated how prior experience (schema) changes the molecular plasticity requirements for new learning. Rats first learned contextual fear conditioning (Task 1), then were tested on a similar second conditioning (Task 2). The key question: does schema change NMDAR requirements for Task 2, and if so, what maintains the schema?

## Key findings

1. **Schema enables NMDAR-independent learning** -- second conditioning does not require dorsal hippocampal NMDARs if the first-task memory is still hippocampus-dependent and a schema exists.
2. **Procedural similarity is what matters** -- schema effects depend on Task 1 and Task 2 sharing procedural structure, not just prior context or shock exposure.
3. **ACC is causally required for schema maintenance** -- disrupting ACC after Task 1 reinstates NMDAR-dependency for Task 2. The cortical prior must be actively maintained.
4. **Hippocampal dependency of Task 1 is required** -- when Task 1 memory becomes hippocampus-independent, schema effects disappear. Prior must be accessible to hippocampus.

## Relevance to INV-044

Provides causal evidence for the prior-before-posterior necessity:
- **Schema = prior structure** maintained in cortico-hippocampal circuit
- **NMDAR-independent slot-filling = efficient posterior inference** when prior is available
- **ACC disruption = prior removal** -- posterior inference collapses back to prior-absent (NMDAR-dependent, full plasticity) mode

When the prior is actively maintained, new instances are rapidly assimilated (slot-filled) without requiring full plasticity. When it is disrupted, the system behaves as if no prior exists -- attribution of new evidence to context slots becomes slow and resource-intensive, consistent with INV-044's prediction that a degenerate prior makes attribution uninformative regardless of training duration.

## Limitations for REE mapping

- Prior is maintained as a persistent trace, not specifically installed during a SWS phase -- the SWS mapping requires an additional interpretive step
- Fear conditioning domain; transfer to REE causal attribution is structural/analogical
- No direct test of what happens when prior construction and slot-filling are attempted simultaneously

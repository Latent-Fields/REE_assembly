# Bouton 2004 — context as a categorical retrieval cue, the discrete cousin of V_s

Bouton's 2004 Learning & Memory review is the canonical synthesis of the contextual-renewal phenomenon: extinction does not destroy the original conditioned response but generates a new, context-specific inhibitory association. When an animal is trained in context A, extinguished in context B, and then tested in either A (ABA renewal) or a novel context C (ABC renewal), the extinguished response reappears. Context functions as a retrieval cue that selects which of multiple associations — the original CS-US or the extinction CS-(no-US) — is currently in force. Extinction memories are documented to be more context-dependent than acquisition memories, which generalise more broadly. Bouton's framework also covers spontaneous recovery (effectively, time as a context dimension), reinstatement (presenting unsignalled USs after extinction), and rapid reacquisition.

This entry is relevant to REE's V_s machinery (MECH-269 verisimilitude scalar, MECH-287 staleness propagation) because Bouton's contextual renewal is the closest behavioural cousin to per-region trust modulation. REE proposes that the agent carries a freshness signal indexed by region, and that this signal modulates trust in rules associated with that region. Bouton demonstrates that biological agents do effectively maintain per-context "which of these competing associations is currently active" decisions: the same CS does not retrieve the same response in different contexts. The structural premise — that biological learning systems do per-context selection of which knowledge to act on — is well-established here. The Bouton review is a useful biological grounding for the substrate-availability premise of MECH-269.

The genuine REE divergence is in the form of the per-region signal. Bouton's framework is categorical and retrieval-based: context A vs context B vs context C are discrete cues that select among discrete associations via a memory-retrieval process. REE's V_s is continuous and trust-modulation-based: the agent maintains a graded scalar per region indicating how trustworthy current beliefs about that region are, and this scalar modulates rule-arbitration weights in a continuous fashion. The two are clearly cousins — both are forms of "the agent's response to a given input depends on per-region/per-context state about that region/context" — but they are not interchangeable. The continuous-scalar form REE proposes has not been directly demonstrated in the Bouton tradition; it would require a different experimental paradigm (graded context manipulations, or measurement of internal trust scalars rather than behavioural retrieval).

There is a small but interesting body of work that bridges Bouton's categorical context with REE's continuous freshness — Gershman, Niv and colleagues have proposed Bayesian latent-state models in which "context" is a continuous-valued posterior over which discrete latent state the agent is currently in, and that posterior modulates retrieval weights in a continuous fashion. This is meaningfully closer to V_s than Bouton's framework alone, although it still does not have the per-region scope REE proposes (a single agent maintains many per-region V_s scalars in parallel). For governance purposes, the right characterisation is that REE's V_s has substantial biological support for the structural premise (per-context/per-region modulation of which rules apply is real), with the continuous-scalar and per-region-broadcast extensions being REE-specific.

The functional caveat worth recording is that Bouton's framework is principally about which memory is retrieved when responding to a CS, whereas REE's V_s is about how much trust to place in a rule when arbitrating among competing rules. These are distinguishable processes — retrieval can succeed (the agent recalls the rule) while trust modulation reduces the rule's weight in arbitration. REE's V_s sits more naturally in the arbitration stage than in the retrieval stage. The Bouton-style evidence supports the existence of per-context state but does not specifically support the arbitration-stage routing REE proposes. For this entry's R3 verdict, supporting direction with confidence 0.65, with the caveat that the continuous-scalar arbitration-routing is REE-novel.

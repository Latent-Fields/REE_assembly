# Pessoa & Adolphs 2010 — affective evaluation as distributed network, not single channel

The Pessoa & Adolphs 2010 review is principally an argument against a once-dominant model of amygdala function: the so-called "low road" account in which a fast subcortical pathway through superior colliculus and pulvinar to amygdala mediates non-conscious affective processing of visual stimuli, with cortex playing a secondary role. The authors marshal anatomical, lesion, and human imaging evidence to argue that the low-road framework is empirically wrong: the human pulvinar lacks the pulvinar-amygdala projections required, lesion data is inconsistent with the low-road account, and cortical contributions to affective evaluation are larger than the older framework allowed. The replacement framework is "many roads": the amygdala coordinates multiple cortical and subcortical pathways that evaluate biological significance along multiple dimensions — salience, valence, ambiguity, unpredictability, threat probability, reward probability.

This entry's relevance to REE's MECH-312 dual-stream affective-vs-discriminative arbitration is partial-and-complicated. The supporting half: Pessoa-Adolphs reinforces the structural premise that affective evaluation is not just an output of the same sensory-discriminative pipeline that does identification — there are multiple parallel evaluation routes, and at least some of them are dedicated to biological-significance dimensions that have no cleanly-mappable analog in the discriminative-identification stream. So the basic REE premise that z_harm_a (affective harm stream) carries content that z_harm_s (sensory-discriminative harm stream) does not is consistent with the Pessoa-Adolphs framework.

The complicating half is that Pessoa-Adolphs reframes the amygdala from "the affective output channel" to "the coordinator of a distributed affective-evaluation network". This matters for REE because the cleanest version of MECH-312-d would treat z_harm_a as the readout of an amygdala-localised affective stream, with the amygdala doing the affective work. Under Pessoa-Adolphs, z_harm_a would more accurately be the readout of an amygdala-coordinated cortical-subcortical evaluation network, with the amygdala doing the integration but the actual evaluation distributed across multiple cortical regions. That distinction is not merely cosmetic: it affects how REE substrate would model SD-011 (point-source vs distributed-network) and how MECH-312's combination rule would have to handle uncertainty-modulation of affective input (per-pathway weighting becomes possible if the affective channel is itself a network rather than a single source).

The Pessoa-Adolphs framework also surfaces a dimension that REE has not yet addressed cleanly: salience-vs-valence dissociation. The amygdala-coordinated evaluation network codes salience (how attention-grabbing a stimulus is, regardless of valence sign) somewhat separately from valence (positive vs negative). REE's affective harm stream conflates these — it carries the "this is harm" content with implicit high salience and negative valence. A more biologically faithful version of SD-011 would dissociate the two, which would expand z_harm_a from a scalar into a {salience, valence} tuple. Whether that expansion is worth the substrate cost is a V3 vs V4 question.

For the R2 verdict in the synthesis, this entry is a SUPPORTS-WITH-CAVEATS for the structural premise of MECH-312-d, with the caveat that the cleanest two-stream picture is harder to defend after Pessoa-Adolphs. REE retains the architectural option to treat z_harm_a as a clean stream for engineering reasons (with a documentation note that the biological substrate is more distributed), or to absorb the distribution into a richer SD-011 substrate. Either way, the Pessoa-Adolphs evidence is a constraint on the mapping rather than an architectural contradiction. Confidence is set at 0.68 (supports) with the moderate-mapping-fidelity flag.

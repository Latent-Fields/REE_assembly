# Koechlin & Summerfield 2007 — Information theoretical approach to prefrontal executive function

According to PubMed, Koechlin & Summerfield (2007, [Trends Cogn Sci 11:229-35](https://doi.org/10.1016/j.tics.2007.04.005)) propose an information-theoretical theory of executive control implemented in lateral prefrontal cortex. The architectural commitment is that executive control operates via hierarchically-ordered control signals organised along the anterior-posterior axis: posterior PFC handles sensorimotor mappings, mid-lateral PFC handles context-conditional rules, and rostral PFC handles increasingly abstract levels (episode, branching, prospective control). The theory clarifies how executive control can operate as a unitary function despite the requirement that information be integrated across multiple distinct, functionally specialised prefrontal regions — the integration happens via the gradient itself, with each level providing contextual conditioning to the levels below it.

For SD-033e Prong D, this paper provides the architectural framing within which the FPC analog sits. The FPC analog is at the rostral end of the executive-control gradient — at the most abstract / branching / counterfactual level. Translated to REE: the FPC analog operates at a higher level of abstraction than SD-033a (lateral PFC analog, which handles rule-conditional bias) and SD-032a (SalienceCoordinator operating mode signal), specifically handling branching / counterfactual / competing-goal level control that lies *above* rule application. The implementation-level recommendation that follows is that the FPC analog should not be wired in parallel with SD-033a but should sit hierarchically above it, with FPC outputs modulating SD-033a's rule_state when goal hierarchies shift.

I want to flag two caveats. The first is that the strict A-P gradient framing has been challenged by subsequent work — notably Badre & Nee 2018 — which argues for a more graded / integrated organisation rather than a strict cascade. SD-033e should not lock to a strict A-P localisation as if it were a decided architectural fact. The safer commitment is hierarchical-but-not-strictly-cascaded: the FPC analog operates at a higher level than SD-033a and modulates it, but the modulation is bidirectional and integrated rather than purely top-down.

The second caveat is the discrete-vs-continuous question. Koechlin & Summerfield's gradient is described as continuous — each level handles a specific kind of contextual conditioning, but the levels are organised along a smooth axis. SD-033e's operating-mode commitment (inheriting from MECH-261) is discrete. The mapping requires the assumption that operating modes correspond to discrete bins of an underlying continuous gradient. This is a workable simplification, not a derived prediction. If the V4 elaboration of SD-033 governance benefits from a more graded mode-state representation, Koechlin's framework would support that direction.

# The Sources of Normativity: Young Children's Awareness of the Normative Structure of Games

**Rakoczy, Warneken, & Tomasello (2008) -- Developmental Psychology, 44(3), 875-881**
DOI: 10.1037/0012-1649.44.3.875

## What the study did

Rakoczy and colleagues introduced 2- and 3-year-old children to simple rule games with specific, arbitrary rules about how objects should be used. A puppet then joined and performed an action that either (a) constituted a normative violation within the game or (b) was the identical physical action performed in a control context outside the game. Children's responses -- protest, critique, or teaching -- were scored and compared.

## Key findings

Both age groups produced significantly more normative responses to violations in the game context than to the same actions outside it. Three-year-olds showed more explicit normative language ("That is not right," "One mustn't do it like that"), while 2-year-olds showed the sensitivity at a less explicit level. Crucially, the violation of the identical action was only a violation because the game frame was active -- the context tag determined normative status. Children responded to third-party violations, not just violations against themselves, suggesting they understood rules as agent-neutral, applying to anyone who participates in the practice.

## REE relevance

This finding maps directly to Q-035 (minimal frame signal architecture) and ARC-049 (co-maintained context tag). The fact that an action's normative status depends on whether the frame is active -- and children can track this -- demonstrates that the ARC-049 frame is not merely an open/close authorization tag but carries structured content: these specific constraints apply within this frame. The real-time detection of violations implies that game state relative to the frame is being monitored during play, not just at boundaries. The bilateral enforcement finding (children policing third-party violations) directly supports INV-059: mutual frame maintenance is an active shared responsibility, not passive co-presence.

## Limitations and caveats

The paradigm cannot distinguish continuous monitoring from reactive computation at the moment of action. Whether children actively monitor game state between actions (heartbeat) or only sample the frame at salient events (reactive) is not separable from this design. Games were adult-introduced, so ecological validity for naturalistic peer-constructed play is limited.

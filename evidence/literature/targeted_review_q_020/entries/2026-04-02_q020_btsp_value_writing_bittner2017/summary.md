# Bittner, Milstein, Grienberger, Romani & Magee (2017) — BTSP: The Write Mechanism for Value-Shaped Hippocampal Geometry

**Claims tested:** Q-020, MECH-073, ARC-007, MECH-074
**Evidence direction:** mixed | **Confidence:** 0.76

---

## What the paper did

Bittner and colleagues asked a fundamental mechanistic question: how do CA1 place fields form? The textbook answer — Hebbian coincidence detection — predicts that synaptic potentiation should require near-simultaneous presynaptic and postsynaptic firing. This predicts that place fields should require multiple visits to a location before potentiation accumulates. But single-trial place field formation is observed, and Hebbian mechanics are insufficient to explain it.

The paper's solution: a non-Hebbian rule they named behavioral time scale synaptic plasticity (BTSP). Using in vivo dendritic recordings in head-fixed mice combined with in vitro slice validation, they showed that calcium plateau potentials in CA1 dendrites potentiate inputs that arrived *seconds* before and after the plateau — far outside the millisecond window of Hebbian STDP. One plateau event can create a place field in a single trial. The plateau acts as a write event; anything that was active in the seconds before and after the plateau gets potentiated. The result is a place field centered slightly behind the plateau location, with an asymmetric temporal profile consistent with the observed seconds-long plasticity window.

## Key findings relevant to Q-020

BTSP resolves a key mechanistic question in the Q-020 debate: *how* does value get written into hippocampal geometry? The answer, from Bittner et al., is: an external trigger (the plateau potential) initiates a write event that potentiates recent input history. The plateau is not generated by CA1 computation — it is driven by external inputs converging on CA1 dendrites. In a naturalistic setting, the events most likely to trigger plateau potentials are behaviorally salient: unexpected rewards, threats, strong prediction errors. These events are precisely the ones that carry value signals from VTA (dopamine), amygdala (BLA→CA1 projections), and entorhinal cortex.

This has a direct implication for Q-020. BTSP describes the mechanism by which a MECH-074-type amygdala write head could inscribe value into hippocampal geometry: BLA or VTA inputs trigger plateau potentials in CA1 dendrites, which then potentiate the preceding spatiotemporal sequence of entorhinal and CA3 inputs. The result is a place field — or, by extension, an abstract state representation — that is geometrically anchored to a location associated with a value-relevant event. The hippocampus did not compute the value; the BLA/VTA told it where to write, and BTSP did the writing.

## REE translation: BTSP supports Resolution A for Q-020

This is the mechanistic key to Q-020 Resolution A. ARC-007's "no value computation" constraint survives if the hippocampus is understood as a BTSP-writing substrate: external value signals (VTA dopamine, BLA calcium-triggering projections) determine *when and where* plateau events fire; BTSP then writes the surrounding synaptic input into long-term geometry. The hippocampus stores the geometry but does not evaluate it. Value computation happens upstream (VTA, amygdala, OFC); value geometry storage happens in CA1 via BTSP.

For MECH-074 (amygdala as read/write head for hippocampal map): the write mechanism is now mechanistically specified. BLA axons project to CA1 distal dendrites and can trigger calcium events that initiate BTSP. This is not hypothetical anatomy — BLA→CA1 dendritic projections are well established. The amygdala write head in MECH-074 is, mechanistically, a BTSP-triggering pathway.

## Limitations

The paper demonstrates BTSP for spatial place fields in head-fixed mice with highly controlled dendritic recording conditions. Whether BTSP operates identically in freely-moving animals, and whether it extends to abstract non-spatial representations (e.g. the abstract value space of Knudsen & Wallis 2021), requires inference. The identity of the endogenous plateau-triggering inputs in naturalistic foraging or social contexts has not been established — it is plausible that BLA and VTA projections serve this role, but direct optogenetic confirmation targeting those pathways specifically has not been done (as of 2017; subsequent work has made progress).

The paper's evidence direction is coded as "mixed" because it simultaneously supports MECH-073 (showing how value gets into the geometry — a fact MECH-073 depends on) and supports Resolution A for Q-020 (showing that the writing agent is external, not hippocampal computation). Neither a clean weakening of ARC-007 nor a clean support of it; rather, it specifies the mechanism that makes both MECH-073 and Resolution A coherent at the same time.

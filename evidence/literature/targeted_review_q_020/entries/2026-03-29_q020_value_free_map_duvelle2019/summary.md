# Duvelle et al. (2019) — Insensitivity of Place Cells to the Value of Spatial Goals

## What the paper did

Duvelle and colleagues designed a two-choice navigation task in which rats had to choose between two unmarked goal locations offering different reward magnitudes (1 vs. 3 pellets). The locations were unmarked, requiring genuine spatial memory for value-differentiated goals. They recorded dorsal CA1 tetrodes throughout and asked whether place cell firing properties -- rate, spatial selectivity, field placement -- varied as a function of goal value. Critically, they also checked for "out-of-field firing" at goal locations, which prior work had associated with goal-relevant modulation.

## Key findings

Rats were clearly sensitive to goal value behaviorally, choosing the higher-value goal significantly above chance. Yet dorsal CA1 place cells showed no modulation by reward magnitude -- firing rates, spatial selectivity, and field locations were equivalent whether the goal was low or high value. There was detectable out-of-field firing at goal locations (consistent with prior literature), but this was not differentiated by value. The authors conclude that dorsal hippocampus encodes a "value-free spatial representation" that links to reward value via downstream circuits rather than encoding value within its own spatial map.

## REE translation

For Q-020, this paper is the primary empirical support for ARC-007's premise: the hippocampal module navigates terrain without computing value anew. If dorsal CA1 place cells genuinely do not carry reward-magnitude information, then the residue field R(x,t) could be geometrically structured by spatial relationships while value is encoded in a separate layer or circuit. The clean behavioral-versus-neural dissociation is exactly the kind of evidence ARC-007 implicitly assumes.

Importantly, this is not incompatible with Jimenez (2018) or Knudsen and Wallis (2021). Jimenez found valence-encoding in *ventral* CA1; Knudsen and Wallis used abstract value in primates. Duvelle's dorsal CA1 finding can be true simultaneously. If so, REE's hippocampal module may need to be conceived as a dorsal-CA1-like spatial navigator whose outputs are modulated by a ventral-CA1-like affective layer -- the two coexist rather than conflict.

## Limitations and honest caveats

The null result is the critical epistemic limitation: no reward-magnitude coding in dorsal CA1 does not mean no value coding anywhere in hippocampus, nor does it rule out harm-residue coding of a different kind. The task manipulates reward magnitude, not aversive or harm-related valence -- the generalization to REE's harm context is an additional inferential step. The dorsal CA1 specificity means this result cannot be used to claim hippocampus as a whole is value-free; only that one subregion is value-insensitive to one class of value (spatial reward magnitude).

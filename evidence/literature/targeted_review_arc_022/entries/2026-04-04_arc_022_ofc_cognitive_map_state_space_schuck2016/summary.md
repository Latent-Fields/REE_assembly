# Summary: OFC as Cognitive Map of State Space — Schuck, Cai, Wilson & Niv (2016)

**Claim tested:** ARC-022 (Hierarchical effect-object abstraction pipeline)
**Evidence direction:** supports
**Confidence:** 0.68

ARC-022 assigns a specific functional role to the Goal/Avoid layer: it consumes narrative simulations from the DMN-regime and produces a valenced possibility space -- a structure encoding which trajectories are approach-relevant and which are avoidance-relevant. The question is whether any brain region plausibly plays this role. Schuck and colleagues provide the most direct empirical evidence that orbitofrontal cortex (OFC) encodes exactly this kind of abstract state object.

Their participants performed a 16-state decision task in which task states were not directly observable from sensory input -- they had to be inferred. fMRI pattern classification showed that OFC activity patterns reliably encode which abstract task state the agent currently occupies. Critically, decoding accuracy correlates with task performance and with the occurrence of individual behavioral errors. This is not mere state encoding; it is state encoding that is doing functional work in the pipeline. When OFC loses track of which abstract state the agent is in, behaviour degrades.

In REE terms, this is the Goal/Avoid layer operating as claimed: it maintains an abstract state object (which region of the valenced possibility space is currently occupied) that is upstream of specific action selection. The OFC cognitive map is not a sensory representation -- it is an abstraction over task history and inferred state structure, exactly the kind of typed object that ARC-022's Goal/Avoid layer would receive from upstream narrative simulation. The fact that OFC state decoding accuracy tracks behavioral error directly links the typed-object representation to the functional role ARC-022 assigns this layer.

The main gap is that Schuck et al. characterise OFC in isolation. They do not establish that OFC receives its state-map representations from a DMN-regime narrative simulation process, nor that it sends typed output to a hippocampal navigation layer. The pipeline context is inferred rather than tested. There is also a relevant distinction between the OFC state map (encoding where in the state space the agent is now) and the Goal/Avoid possibility space (encoding which trajectories lead toward desirable vs. aversive outcomes). These may be the same computation in different terms, or they may be adjacent but distinct computations. The paper supports one pole of this; whether OFC also generates the valenced possibility space over future trajectories (rather than just tracking current position) would require additional evidence.

*Source:* Schuck NW, Cai MB, Wilson RC, Niv Y (2016). Human Orbitofrontal Cortex Represents a Cognitive Map of State Space. *Neuron* 91(6):1402-1412. doi:10.1016/j.neuron.2016.08.019

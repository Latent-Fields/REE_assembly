# Hazy, Frank & O'Reilly (2007) — Towards an Executive Without a Homunculus

**Q-019 relevance: strongest direct support for parallel multi-loop BG architecture**

Hazy, Frank, and O'Reilly (2007) situate the PBWM model within an evolutionary-computational framework, arguing that the BG's role as a dynamic gate on PFC is an evolutionary extension of its motor control function. The same striatum-SNr-thalamus-frontal disinhibition circuit operates in anatomically parallel loops across motor, cognitive, and limbic/motivational domains. All loops share the same canonical form; each loop runs independently with its own BG sub-module. Dopaminergic reinforcement trains gating policies in each loop separately.

Key mechanisms:
- **Motor loop**: Basal ganglia gate motor action selection (direct pathway facilitates; indirect suppresses).
- **Cognitive loop (PFC)**: Basal ganglia gate working memory maintenance across independently-gated PFC stripes, enabling executive function without a homunculus.
- **Limbic/motivational loop**: Amygdala and ventral striatum contribute Pavlovian affective signals that bias BG learning toward motivationally relevant gating policies.

**Three-gate mapping for Q-019:**
- Motor BG loop → **action commitment gate** (motor execution authority)
- PFC cognitive BG loop → **thought loop gate** (E3 complex WM and planning)
- Sensory/attentional input filtering → **sensorium gate** (what enters the pipeline for E3 attention)
- Amygdala/ventral striatum affective input → **hippocampal affective residue** shaping sensorium gate weighting

This paper provides the clearest literature statement that the three gates are functionally distinct instances of a canonical BG circuit operating in anatomically parallel loops — directly supporting Q-019's three-gate architecture.

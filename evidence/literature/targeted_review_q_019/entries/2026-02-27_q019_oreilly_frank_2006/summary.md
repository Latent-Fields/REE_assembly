# O'Reilly & Frank (2006) — PBWM: Making Working Memory Work

**Q-019 relevance: supports three-gate BG architecture (sensorium, thought-loop, commitment)**

O'Reilly and Frank (2006) present the PBWM (Prefrontal Basal Ganglia Working Memory) model, a computational account in which the basal ganglia act as the executive controller of prefrontal working memory via dopamine-trained Go/NoGo reinforcement learning. The core circuit is a striatum-SNr-thalamus-PFC disinhibition loop: direct-pathway striatal neurons inhibit the tonically active SNr, releasing thalamic suppression and enabling thalamocortical reverberation in PFC deep layers, effectively "opening the gate" on a specific PFC memory stripe.

The model specifies two functionally distinct gate types in parallel:
- **Input gating**: BG decides when to allow new information into PFC deep-layer maintenance.
- **Output gating**: A separate BG circuit determines which maintained PFC representation is released downstream to drive action selection.

PFC is organized into independently-gated "stripes," each with its own striatal module. Dopaminergic reinforcement signals (bursts = Go, dips = NoGo) train the gating policies.

**Three-gate mapping for Q-019:**
- Input gating → **sensorium gate**: what enters the loop
- PFC stripe maintenance → **thought loop gate**: E3 complex operations across multiple WM slots
- Output gating → **action commitment gate**: BG releases representation to drive motor execution

The PBWM model treats all three gates as instances of the same canonical BG disinhibition circuit. REE adds oscillatory specificity (beta/theta) and anatomical separation (STN vs SNr pathways). The paper does not address the hyperdirect pathway.

# Buckner, Andrews-Hanna & Schacter (2008) — The Brain's Default Network

**Q-019 relevance: anatomical grounding for the thought loop substrate (mixed/indirect support)**

Buckner, Andrews-Hanna, and Schacter's landmark review synthesizes evidence that the brain's default network is organized for internal simulation rather than external perception or motor execution. The network is minimally connected to primary sensory/motor cortices and strongly anticorrelated with externally-directed attentional systems.

**Network anatomy:**
- **Medial temporal lobe subsystem**: Hippocampal formation, parahippocampal cortex, retrosplenial cortex — provides associative memory content as building blocks for simulation.
- **Medial prefrontal subsystem**: vmPFC, dmPFC — constructs self-relevant simulations from memory content.
- **Integration hubs**: Posterior cingulate cortex and inferior parietal lobule.

**Functional profile:**
- Active during autobiographical retrieval, prospective planning, social cognition, mind-wandering.
- Deactivates during externally-directed task engagement.
- Anticorrelated with frontoparietal control networks (task-positive systems).

**Three-gate mapping for Q-019:**
- Default network → **thought loop substrate**: the system that E3 complex's internal simulation, trajectory planning, and affective weighting operate within. Medial temporal subsystem = hippocampal map function; mPFC subsystem = commitment-gating and self-relevant evaluation.
- Default network deactivation during external cue → sensorium gate opening **interrupts** thought loop (anticorrelation supports loop exclusion requirement).
- Default network deactivation before voluntary action → action commitment gate activation **suppresses** thought loop (simulation-to-execution transition involves default network suppression).

**Mapping caveat**: This paper does not address BG gating mechanisms and makes no claims about oscillatory control planes. Evidence direction is "mixed" — strongly supports the existence of the thought-loop anatomical substrate, but does not speak to the BG loop structure of Q-019 directly.

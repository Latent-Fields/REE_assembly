## Research Question

Is there evidence that hippocampal sharp-wave ripples (SWRs) during waking quiescence (not sleep) serve map consolidation / place field updating, distinct from goal-directed prospective planning?

## Key Questions Addressed

1. Evidence for waking (non-sleep) hippocampal replay consolidating the cognitive map -- YES (O'Neill 2010, Krause 2021)
2. Evidence that waking replay is distinct from goal-directed prospective sweeps -- PARTIAL (Krause 2021 momentum finding; state-dependency in McHugh 2023)
3. What triggers consolidation mode (quiescence, absence of active goal) -- INDIRECT (state-dependent SWR circuitry, McHugh 2023)
4. Waking consolidation necessary for subsequent goal-directed performance -- YES, causal evidence (Ego-Stengel 2010)

## Summary Assessment

All 4 entries support ARC-038. The evidence package covers:
- Causal necessity of waking-rest SWR replay for subsequent spatial performance (Ego-Stengel)
- Ubiquitous trajectory-with-momentum content in awake SWRs, distinct from sleep replay (Krause)
- Waking rest as an established site of consolidation-oriented SWR activity (O'Neill)
- State-dependent SWR circuitry providing mechanistic grounding for mode-switching (McHugh)

**Main gap remaining:** No paper directly tests the goal-directed-vs-consolidation distinction as a clean manipulable switch (e.g., presence vs. absence of active z_goal). The distinction is supported by convergent indirect evidence but not a single clean experiment.

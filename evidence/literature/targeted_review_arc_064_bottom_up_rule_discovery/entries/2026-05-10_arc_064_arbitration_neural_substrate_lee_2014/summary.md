# Lee, Shimojo & O'Doherty 2014 -- Neural computations underlying arbitration between MB and MF learning

According to PubMed ([DOI](https://doi.org/10.1016/j.neuron.2013.11.028), PMID 24507199), Lee, Shimojo and O'Doherty took the Daw 2005 normative arbitration proposal and put it through human fMRI. The empirical question is sharp: if the brain follows uncertainty-weighted arbitration, then somewhere in the brain there should be (i) reliability signals for each of the candidate behavioural systems, (ii) a comparison signal that takes those reliabilities as input and produces an arbitration decision, and (iii) some downstream mechanism that implements the decision. They find all three. Inferior lateral PFC and frontopolar cortex encode both reliability signals and the comparison-between-them. The effective connectivity from these prefrontal regions to model-free valuation areas is negatively modulated by the degree of model-based control -- when the arbitrator says 'use the deliberative system,' the prefrontal output suppresses the habit valuation pathway.

For MECH-312 this is the empirical anchor that complements Daw 2005's normative argument. The Daw paper says the brain *should* arbitrate this way; the Lee paper says the brain *does* arbitrate this way, with identifiable neural substrates. For REE's substrate-design decision the implication is concrete: MECH-312's arbitration substrate has to (i) compute per-channel reliability signals, (ii) compute a comparison, and (iii) modulate the dominant channel's downstream output. Each of these is a substrate-level commitment; the lateral-PFC anchor in Lee 2014 supplies the canonical biological exemplar.

There is also an architectural detail in Lee's findings worth carrying into the synthesis. The arbitration mechanism Lee identifies works by modulating effective connectivity to the model-free valuation system, not by switching between channel outputs at a downstream gate. That is, the channels are always running; arbitration changes how strongly each one's output reaches action selection. This is a different architectural choice from the alternative (binary channel-switching) and aligns with Pull 1's R4 verdict that ARC-065 always-on diversity is in continuous-presence mode with triggered dominance. The same logic likely applies to ARC-062 and ARC-064 -- both channels are always producing rule labels and confidence estimates; arbitration determines which set of labels gets weight in the policy.

The mapping caveat: Lee's task tests MB-vs-MF arbitration explicitly. REE's ARC-062 / ARC-064 / ARC-065 are not strict MB/MF analogs; they sit on a different axis (top-down-instructed vs bottom-up-extracted vs always-on-diverse). The neural-substrate claim (lateral PFC + frontopolar host the arbitration computation) is plausibly general -- the lateral PFC is a sensible substrate for any of these arbitration variants -- but the specific contrasts Lee tests do not directly cover REE's arbitration scope.

Confidence 0.80. Neuron venue + methodologically clean fMRI. Mapping fidelity moderate-high for the substrate-level claim. Transfer risk slightly elevated because the MB-vs-MF axis is not a strict analog of REE's arbitration scope.

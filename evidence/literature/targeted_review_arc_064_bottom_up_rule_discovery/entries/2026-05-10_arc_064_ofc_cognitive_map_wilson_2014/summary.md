# Wilson, Takahashi, Schoenbaum & Niv 2014 -- OFC as cognitive map of task space

According to PubMed ([DOI](https://doi.org/10.1016/j.neuron.2013.11.005), PMID 24462094), Wilson and colleagues offer a unifying theory of orbitofrontal cortex function that has set the agenda for the OFC literature ever since. Their proposal is computationally clean: OFC does not compute value, does not store stimulus-outcome associations, does not represent reward magnitude. It does something subtler -- it labels the current task state. That is, OFC provides a discrete (or near-discrete) categorical handle on 'where the agent currently is' in some abstract task-structure space, and downstream reinforcement-learning machinery uses this label to query the right value, the right policy, the right expected outcome. Critically, OFC's role becomes load-bearing precisely when the current state is not directly observable from the sensory input -- reversal learning where the contingency has flipped but no sensory cue marks the flip; delayed alternation where the correct action depends on what was done last trial; devaluation where the outcome value has changed but the stimulus has not. Under all these, OFC lesions disrupt behaviour because the agent loses the latent-state abstraction.

For ARC-064 this is the second load-bearing R2 paper, but with a complication. ARC-064's bottom-up rule discovery extracts behavioural patterns and encodes them as self-generated rule labels for downstream policy heads to consume. Wilson et al. propose that OFC hosts exactly the kind of abstract state-label substrate that bottom-up rule discovery needs to write to. Their framework unifies a sprawling clinical and animal literature under a single computational claim, and that claim plays nicely with REE's architecture: an abstraction layer between sensory experience and policy that holds rule-equivalent labels.

The mapping caveat I want to flag carefully. Wilson and Niv's task states are defined by the experimenter -- the structure of the task imposes the state taxonomy, and OFC's job is to figure out which task-imposed state it is currently in. ARC-064, by contrast, posits that the agent itself extracts the state taxonomy by clustering its own behaviour. These are subtly different operations. The first asks: 'given this state structure, which state am I in now?' The second asks: 'is there a state structure here at all, and if so, what are the categories?' OFC may do both; OFC may do only the first. The Wilson paper does not settle this. Niv 2019 (also in this set) is more explicit that representation learning -- the bottom-up extraction question -- is where the field is heading post-2014, and OFC is implicated in both directions.

What this paper buys for the synthesis: a strong default that ARC-064's extracted-rule substrate maps onto OFC-analog territory, distinct from the hippocampal regularity-extraction pathway (Schapiro 2017) which would be doing the upstream pattern-discovery. A two-stage architecture starts to crystallise -- hippocampal pathway extracts regularities from the trajectory record; OFC-analog pathway labels the regularities and exposes them to the policy. The R2 verdict 'multi-substrate distributed pathway' starts to look concrete: this is not all one mechanism, it is a chain.

Confidence 0.84. Neuron venue and foundational status give high source quality; the mapping fidelity is strong for the substrate-of-state-labels role and weaker for the bottom-up-extraction question that ARC-064 specifically poses. Transfer risk moderate -- the OFC framework is robustly tested in monkey + rodent + human fMRI, but the bottom-up provenance question is not central to the original paper.

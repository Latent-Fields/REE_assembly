# Schuck, Cai, Wilson & Niv 2016 -- Human OFC represents a cognitive map of state space

According to PubMed ([DOI](https://doi.org/10.1016/j.neuron.2016.08.019), PMID 27657452), Schuck and colleagues took the theoretical proposal in Wilson 2014 and ran the empirical confirmation. They built a 16-state decision task, trained human participants on it, scanned them with fMRI, and applied multivariate pattern analysis to ask whether OFC activity carries a decodable signature of the current task state -- particularly when that state cannot be read off the sensory input alone. The answer is yes. OFC patterns decode the current state, decoding accuracy correlates with overall task performance, and decoding errors on individual trials predict the participant's behavioural errors on those trials. Even better, the *similarity* between OFC patterns for consecutive states correlates with behavioural accuracy on the transition between those states -- a graded representational structure, not a categorical lookup table.

For ARC-064 this is the empirical complement to Wilson 2014. If we believe the bottom-up rule discovery pathway, then the rule-states it extracts should generate decodable substrate-level signatures whose decodability correlates with the rule's behavioural utility. Schuck et al.'s methodology is the obvious template for an REE validation experiment: cluster trajectories, extract rule-state labels, train a decoder on the substrate writing those labels, and check whether decoder accuracy predicts policy performance. If it does, the substrate is doing real work. If it does not, the substrate is hosting epiphenomenal labels.

The caveat I want to be candid about: Schuck's participants were trained on the 16 states. The states were defined by the experimenter and learned top-down through instruction and practice. ARC-064's bottom-up rule discovery is the harder case where the agent has to extract the states from its own behaviour without external supervision. The OFC-state-decoding signature is what we would expect to find if the bottom-up extraction works, but the Schuck paper does not directly test the bottom-up direction. The cleanest reading is: this paper supports the architectural claim that OFC-analog substrate hosts state labels in a decodable form, with the bottom-up provenance question deferred.

There is also a substrate-cardinality question lurking. Schuck's task has 16 well-separated states; the decoding works because the states are statistically distinguishable. ARC-064's bottom-up rule discovery, applied to a small monomodal-collapsed REE trajectory record, would have at most one or two extractable states -- not enough for a multi-state decoding test. Implementation-side, this means ARC-064's validation experiment needs an environment where the trajectory record actually contains multiple modes for the bottom-up pathway to find. Cross-reference Pull 1's R3 verdict that ARC-065 is upstream-mandatory: the same point applies here.

Confidence 0.78. Neuron venue and methodologically clean fMRI MVPA. Mapping fidelity moderate because the task is top-down trained, not bottom-up discovered. Transfer risk slightly higher than for theoretical Wilson 2014 because the empirical finding is more specific and the gap between top-down and bottom-up state acquisition is real.

# Yonelinas, Ranganath, Ekstrom & Wiltgen 2019 -- A contextual binding theory of episodic memory: systems consolidation reconsidered

DOI: [10.1038/s41583-019-0150-4](https://doi.org/10.1038/s41583-019-0150-4) -- Nature Reviews Neuroscience (PubMed 30872808)

## What the paper did

A theoretical review reframing episodic memory consolidation. Standard systems consolidation theory (SSCT) holds that hippocampal traces eventually transfer to neocortex over offline periods (sleep). Yonelinas and colleagues argue instead that episodic memory remains hippocampus-dependent throughout life because the hippocampus binds item information to context, and forgetting reflects contextual interference rather than failed transfer. They draw on behavioural, lesion, neuroimaging, and sleep evidence.

## Findings relevant to V_s foundation

Three points carry. (1) The hippocampus is presented as a contextual-binding device throughout the lifespan, not as a temporary buffer. This argues that even old, well-consolidated memories retain hippocampal anchors -- consistent with multiple anchors per place coexisting indefinitely (some highly active, some near-dormant but recoverable under the right context). (2) Forgetting is contextual interference: anchors degrade when context drifts away from the original encoding context, not because the trace itself decays. The implication is that V_s drop on an anchor -- as the substrate plan defines it -- is a measure of contextual mismatch, which is exactly the right signal for routing decisions. (3) Sleep does not transfer information; it reduces interference. This matters for MECH-285 (sleep-priority) and the broader V_s-residual framework: if sleep is reducing interference rather than transferring, the per-anchor V_s should be expected to recover after sleep when context-tagging is sharpened, not because the underlying representation moved.

## Translation to REE / MECH-269 / MECH-272 / MECH-285

For dual-trace at the anchor level, this paper provides theoretical backing: distinct hippocampal traces for the same physical place coexist as a function of distinct contextual binding, and routing across them is a context-dependent retrieval operation. MECH-272 routing then has a clean theoretical home -- the routing is the contextual-retrieval cue selecting which anchor to activate, with V_s providing the mismatch signal that triggers re-routing or anchor-reset.

For MECH-285 (sleep-priority), the contextual-binding view changes what sleep is doing: it is sharpening context-tagging on existing anchors (reducing interference), not moving traces to cortex. The V_s residual that accumulates during waking and gets prioritised for sleep-time processing should be interpreted as a measure of contextual interference per anchor, not a measure of "this needs cortical consolidation." This refines MECH-284 operational definition.

For granularity, the contextual-binding view is silent on the specific scale (place-cell-field vs schema vs event-segment) but argues the relevant operation is binding at whatever scale episodic memory operates. This is consistent with multi-scale architecture but does not specify the default.

## Limitations and caveats

This is a theoretical review, not new empirical evidence -- it reinterprets existing data. The contextual-binding theory remains contested; the standard systems consolidation view has substantial supporters. Some specific predictions (e.g., that hippocampal lesions should impair even very old episodic memories) are mixed in the literature, with semantic vs episodic dissociations cutting across age. The mapping to the V_s framework requires accepting the contextual-binding frame -- which is well-argued but not consensus.

## Confidence reasoning

Source quality very high (Nature Reviews Neuroscience, well-cited authors). Mapping fidelity is moderate -- the architectural translation is theoretically clean but depends on accepting a contested theory. Transfer risk is mainly the theory-vs-consensus question; if standard systems consolidation is correct, the V_s-as-contextual-mismatch reading needs adjustment.

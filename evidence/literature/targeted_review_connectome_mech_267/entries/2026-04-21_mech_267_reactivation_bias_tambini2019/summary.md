# Tambini and Davachi 2019 — Awake reactivation biases cognition (MECH-267)

According to PubMed ([DOI](https://doi.org/10.1016/j.tics.2019.07.008)), Tambini and Davachi review human fMRI evidence for spontaneous reactivation of hippocampal representations during awake post-encoding periods, together with hippocampal-cortical interactions. The review is unified around one empirical observation: reactivation during quiet periods after an encoding experience is predictive of later memory, and — more importantly for MECH-267 — persistent reactivation can bias ongoing perception and cognition. The human fMRI extension of the rodent replay literature makes the phenomenon more directly relevant to architectures (like REE's) that deal with nonspatial, episodic-event representations.

For MECH-267 this review is a mixed-direction entry. On one hand, it supports MECH-267's architectural motivation: hippocampal reactivation content in one context biases cognition in subsequent contexts. This is why mode-conditioning matters — if it didn't, internal-replay content would have no path to contaminate external-task action selection, and MECH-267 would be unnecessary. The human-imaging evidence that reactivation does bias subsequent cognition shows that the contamination pathway is biologically real. In that sense the review supports MECH-267's relevance.

On the other hand, the review does not test mode-conditioning at the level REE requires. Tambini and Davachi's paradigm is "task encoding followed by awake rest followed by subsequent test" — a sequence of distinct contexts, but not a structured four-mode taxonomy. The reactivation-biases-cognition phenomenon is established at the phenomenon level; whether the biasing is gated by operating-mode, hypothesis tags, or some other filter is not resolved. It is compatible with MECH-267's gated architecture, but it is also compatible with an ungated architecture in which reactivation always biases ongoing cognition by default.

The V3-EXQ-465 experiment — intrusive simulation filtering — maps onto this tension directly. That experiment tests whether hippocampal proposals generated under internal_replay mode bias subsequent external_task action selection (as Tambini & Davachi's finding would predict for an ungated architecture) and whether they can bypass MECH-094 write gates (which MECH-267 requires them not to). Tambini & Davachi's review supports the biological plausibility of the bias phenomenon; V3-EXQ-465 will test whether the gating works as MECH-267 prescribes.

A failure signature worth recording: if reactivation-based biasing is a generic low-level mechanism that operates regardless of operating mode, MECH-267's specific mode-conditioning commitment is not the right level of description — the correct description would be 'proposals always bias subsequent cognition, gated only by hypothesis tags'. MECH-267 as written does more than this: it says proposal content itself is mode-conditioned, not just that proposals always propagate and are filtered after the fact. The distinction matters for implementation and for the V3-EXQ-465 diagnostic.

Confidence is set at 0.70. Source quality is high (TiCS review, expert authors). Mapping fidelity is moderate because the review supports reactivation-biases-cognition, which is necessary for MECH-267's architectural motivation but does not directly test the mode-conditioning structure. I record `evidence_direction: mixed` because the review provides both support (biological reality of cross-context bias) and a warning (the bias is not obviously gated by mode in the data, so the gating must be architectural if REE requires it).

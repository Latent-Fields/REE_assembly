# Summary: Loewenstein, Raviv & Ahissar (2021) -- Supervised vs Unsupervised Learning in Perceptual Discrimination

**Entry:** 2026-04-04_mech_153_supervised_vs_unsupervised_discrimination_loewenstein2021
**Claim tested:** MECH-153 -- E1 ContextMemory requires supervised context-labeling objective for differentiated representations
**Evidence direction:** supports | **Confidence:** 0.74

---

The question Loewenstein, Raviv and Ahissar set out to answer is deceptively simple: in a trained perceptual discrimination system, which components are shaped by experience alone and which require explicit feedback? Their answer -- a clean empirical dissection -- is that the two are not merely quantitatively different but qualitatively separable. The first stage, where the system learns the statistical structure of the stimulus space, proceeds unsupervised and is, strikingly, *impenetrable* to feedback manipulation. The second stage, where a binary judgment is applied to the percept built in stage one, is fully supervised: it responds to reward protocols within roughly 40 trials.

This two-stage decomposition maps onto the MECH-153 claim with uncomfortable directness. E1 ContextMemory trained on world-model prediction error is a first-stage system: it learns the distributional structure of states encountered near and far from hazards, and it does so competently -- the LSTM is converging, the prediction errors are decreasing, the representations are well-formed. What the EXQ-181b result shows (cosine_sim=0.9999, 0/3 seeds meeting criterion) is that "well-formed" and "discriminative on the relevant binary partition" are not the same thing. The world-model objective rewards accurate state prediction; it does not reward encoding the hazard-proximate/hazard-distal distinction per se, because those states can look nearly identical in z_world. The system learns the statistics of the trajectory without learning to partition it. That is precisely Loewenstein's first-stage failure mode: representation without binary discrimination.

The supervised component in Loewenstein's framework is what MECH-153 is calling for in the ContextMemory training objective: a gradient that explicitly labels the binary partition and rewards the system for encoding it in a retrievable form. The paper's finding that this component responds within 40 trials is actually encouraging for the implementation: it suggests that once the supervised signal is present, the discriminative structure forms rapidly on top of the pre-existing representation. The scaffold is already there; the labeling signal just has to arrive.

Two caveats deserve mention. The substrate in Loewenstein's study is auditory perceptual cortex, not an LSTM context-memory network -- the unsupervised component is a contraction bias in sensory estimation, which is mechanistically quite different from LSTM hidden-state dynamics trained on prediction error. The structural parallel (unsupervised learning of statistics + supervised binary judgment) is real, but the mechanisms are not homologous. The second caveat is that their supervised signal is external feedback from an experimenter; MECH-153 requires an internally-derived label (hazard_gradient_view.max() threshold). Whether self-derived supervision carries the same gradient quality as external feedback is an open question -- one that EXQ-211 was beginning to probe, with mixed results.

Confidence 0.74: the two-stage framework provides a principled conceptual basis for the MECH-153 claim, and the empirical finding that unsupervised experience cannot produce binary discrimination is directly relevant. The transfer from auditory perception to LSTM context memory is a real epistemic gap that prevents higher confidence.
